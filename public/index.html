Status colors

  • Green — ACTIVE
    Processor active and tile count matches expected.
  • Orange — STANDBY / ONLINE
    Processor reachable, in standby role or tiles above expected.
  • Red — MIXED
    Redundancy state mixed/unhealthy. Needs attention.
  • Red blinking — TILES MISSING
    Active tile count is below expected. Critical.
  • Gray — UNASSIGNED
    No processor assigned to this slot.
